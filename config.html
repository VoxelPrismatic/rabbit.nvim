Configuration
Rabbit is quite advanced, so please read types carefully.

Types
NvimHlColor: string
This is a color string.
  • #RRGGBB
    Hex color string
  • name
    Neovim named color
  • :HlGroupName
    Samples the color of a highlight group. Eg :Normal will sample the foreground, background, or special color of Normal
  • :HlGroupName#prop
    Samples color prop of the highlight group. Eg :Normal#fg will sample the property of the foreground color (fg) of Normal. This way, you can pass the foreground color of one color as the background to another.
NvimHlKwargs
This mirrors vim.api.keyset.highlight, but has some fancy features for dynamic coloring.
~ key type description
colors fg NvimHlColor Foreground color. For the color inheritance, the default property will be fg.
bg NvimHlColor Background color. For the color inheritance, the default property will be bg.
sp NvimHlColor Special color, eg for underlines. For the color inheritance, the default property will be sp.
blend integer Blend factor (opacity) for background color, from 0%-100%
others link string Link/alias to another highlight group
default boolean If set, the existing highlight group will NOT be changed
force boolean If set, all attributes from the existing highlight group will be cleared
styles bold boolean Bold text
italic boolean Italic text
strikethrough boolean Strikethrough text
underline boolean Underline text
undercurl boolean Undercurl text
underdouble boolean Underdouble text
underdotted boolean Underdotted text
underdashed boolean Underdashed text
reverse boolean Reversed text
standout boolean Standout text. Normally displayed as reverse text.

legacy cterm string[] An array of the following styles: bold, italic, strikethrough, underline, undercurl, underdouble, underdotted, underdashed, inverse, reverse, standout, nocombine, altfont. This is a legacy option, and it is recommended you use the above boolean options instead.
